  技能描述（中文和英文）

  Trigger: keyword1, keyword2
---

|------|------|

| 场景2 | 具体说明 |

说明第一步要做什么。

## 示例技能

### 示例 1：文案优化技能

```markdown
---
name: copywriting-optimizer
description: |
  文案优化技能。优化营销文案、广告语、产品描述等。
  触发方式：文案优化、广告语、产品描述
  Trigger: optimize copy, ad copy
---

# 文案优化技能

## 技能说明

优化营销文案，使其更具吸引力和说服力。

## 使用场景

| 场景 | 说明 |
|------|------|
| 广告文案 | 优化广告语，提高点击率 |
| 产品描述 | 优化产品介绍，突出卖点 |
| 社交媒体 | 优化微博/小红书内容 |

## 工作流程

### Step 1: 分析原文

- 识别核心信息
- 找出目标受众
- 确定文案目标

### Step 2: 优化策略

- 添加钩子（hook）
- 使用痛点-解决方案结构
- 加入社会证明

### Step 3: 生成多个版本

- 版本A：激进型
- 版本B：温和型
- 版本C：故事型

## 输出格式

```
原文：...
优化后：
版本A：...
版本B：...
版本C：...
```

## 注意事项

- 保持事实真实性
- 避免夸大其词
- 符合广告法规
```

### 示例 2：数据分析技能

```markdown
---
name: data-analyzer
description: |
  数据分析技能。分析CSV/JSON数据，生成图表和洞察。
  触发方式：数据分析、数据可视化
  Trigger: analyze data, visualize data
---

# 数据分析技能

## 技能说明

分析结构化数据，生成统计报告和可视化建议。

## 使用场景

| 场景 | 说明 |
|------|------|
| 销售数据 | 分析销售趋势 |
| 用户行为 | 分析用户行为模式 |
| A/B 测试 | 分析测试结果 |

## 工作流程

### Step 1: 数据加载

读取用户提供的 CSV/JSON 文件。

### Step 2: 数据清洗

- 处理缺失值
- 统一格式
- 验证数据类型

### Step 3: 统计分析

- 计算基本统计量
- 识别异常值
- 发现趋势和模式

### Step 4: 生成报告

- 关键发现
- 可视化建议
- 行动建议

## 输出格式

```
数据概览：...
关键发现：
1. ...
2. ...
3. ...
可视化建议：...
行动建议：...
```

## 注意事项

- 数据量限制在10万行以内
- 支持CSV和JSON格式
- 复杂分析需要明确指定
```

## 技能开发最佳实践

### 1. 命名规范

- 使用小写字母和连字符
- 名称应该清晰描述功能
- 中文名称和英文名称保持一致

### 2. 描述规范

- 中文描述在前，英文描述在后
- 清楚说明触发方式
- 列出主要使用场景

### 3. 文档规范

- 使用表格展示结构化信息
- 使用代码块展示示例
- 使用步骤说明工作流程

### 4. 测试规范

为每个技能提供测试用例：

```
输入：...
预期输出：...
```

## 技能发布流程

### 1. 本地测试

```bash
opensquilla skills add skills/your-skill.md
opensquilla agent -m "测试你的技能"
```

### 2. 提交 PR

1. Fork OpenSquilla 仓库
2. 创建技能分支：`git checkout -b skill/your-skill-name`
3. 将技能文件添加到 `skills/` 目录
4. 提交 PR

### 3. 技能审核

- 代码质量
- 文档完整性
- 测试覆盖率
- 安全性检查
