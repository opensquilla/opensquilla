# 文案优化技能

## 技能说明

优化营销文案，使其更具吸引力和说服力。专注于提高转化率和用户参与度。

## 使用场景

| 场景 | 说明 |
|------|------|
| 广告文案 | 优化广告语，提高点击率和转化率 |
| 产品描述 | 优化产品介绍，突出核心卖点 |
| 社交媒体 | 优化微博/小红书/朋友圈内容 |
| 邮件营销 | 优化邮件主题和正文，提高打开率 |
| 落地页 | 优化落地页文案，降低跳出率 |

## 工作流程

### Step 1: 分析原文

在优化之前，先理解原文的核心信息：

- 识别核心信息/产品卖点
- 确定目标受众画像
- 明确文案目标（点击/购买/注册）
- 识别当前文案的问题

### Step 2: 应用优化策略

根据文案类型应用对应策略：

**广告文案策略：**
- 使用痛点-解决方案结构
- 添加紧迫感或稀缺性元素
- 使用数字和具体数据
- 加入社会证明（用户数、好评率）

**产品描述策略：**
- 使用特性-利益公式（Feature → Benefit）
- 加入场景化描述
- 使用感官语言
- 强调独特价值主张

**社交媒体策略：**
- 使用钩子开头
- 加入表情符号（适量）
- 使用分段式结构
- 加入行动号召（CTA）

### Step 3: 生成多个版本

提供多个优化版本供选择：

- **版本 A：激进型** - 强调紧迫感，强力 CTA
- **版本 B：温和型** - 故事化，情感共鸣
- **版本 C：专业型** - 数据驱动，权威可信
- **版本 D：短平快** - 简洁有力，适合快速浏览

## 输出格式

```
原文分析：
- 核心信息：...
- 目标受众：...
- 文案目标：...
- 当前问题：...

优化版本：

版本 A（激进型）：
...

版本 B（温和型）：
...

版本 C（专业型）：
...

版本 D（短平快）：
...

优化建议：
- ...
```

## 注意事项

- 保持事实真实性，不夸大产品功能
- 避免使用误导性或欺骗性语言
- 符合广告法和平台规范
- 保持品牌调性一致
- A/B 测试验证效果

## 示例

**输入：**
```
优化这个产品描述：
"我们的咖啡豆来自哥伦比亚，味道很好。"
```

**输出：**
```
原文分析：
- 核心信息：哥伦比亚咖啡豆
- 目标受众：咖啡爱好者
- 文案目标：促进购买
- 当前问题：描述过于简单，缺乏吸引力

优化版本：

版本 A（激进型）：
"🔥 限时！哥伦比亚顶级咖啡豆，前100名下单享8折！"
"3天喝出差异，不好喝全额退款！"

版本 B（温和型）：
"每一次冲泡，都是一次哥伦比亚之旅。
我们的咖啡豆来自海拔2000米的安第斯山脉，
手工采摘，48小时内新鲜烘焙。
让清晨的第一杯，成为你一天中最期待的时刻。"

版本 C（专业型）：
"哥伦比亚考卡山谷产区，SCA 85+评分，
100% 阿拉比卡豆，中度烘焙。
杯测笔记：焦糖、柑橘、巧克力尾调。
已通过2000+咖啡爱好者验证。"

版本 D（短平快）：
"哥伦比亚直供 · 新鲜烘焙 · 香浓醇厚
现在下单，次日送达"

优化建议：
- 建议使用版本 B 用于品牌故事页面
- 版本 A 适用于促销活动
- 版本 D 适用于移动端商品卡片
```
